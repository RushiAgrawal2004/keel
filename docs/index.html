Keel

Architecture regression tests for AI-generated code, powered by Graphify.

What Keel Does

Keel reads Graphify's repository graph, discovers architectural invariants, asks humans to approve useful contracts, and then enforces those contracts in CI.

Quickstart

pip install -e ".[dev]"
graphify .
keel discover . --write
keel approve ui_never_touches_database .
keel check . --json --html
keel dashboard .

Core Commands

  • keel discover proposes architecture contracts.
  • keel approve moves a proposal into .keel.yml.
  • keel check blocks approved contract regressions.
  • keel baseline records known existing violations.
  • keel dashboard writes a local HTML dashboard.
  • keel graph-quality explains graph confidence and gaps.

ADR Compiler

Use docs/adr/*.md frontmatter to turn architectural decisions into reviewable Keel contracts. The included docs/adr/0001-ui-database-boundary.md file shows the expected format.

Automation

GitHub Actions runs tests, optional Keel checks, PR comments, docs deployment, and PyPI publishing workflows.
